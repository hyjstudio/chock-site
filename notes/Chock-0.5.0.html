楔子 0.5.02026-07-20

录屏能暂停、恢复，也能选择录进你的声音。输出使用真正的 MP4 容器；系统音与麦克风混成一条音轨，发到微信、飞书等常见接收端也能直接听见旁白。

新功能

  • 可选麦克风——录制前逐次决定，默认关闭；只有主动打开时才申请权限，拒绝后可直达系统设置或不用麦克风继续录制
  • 全链路键盘可达——剪贴板窗口、粘贴面板、截图标注、录屏和面板工具栏都补齐键盘操作
  • 和弦多选与连续粘贴——剪贴板历史支持 Command 和弦多选，图片等内容按顺序逐条自动粘贴

改进

  • 蓝牙耳机或 USB 声卡热插拔时会重建麦克风采集；失败则降级为无麦克风继续录制，不中断当前文件
  • 0.4.9 未曾公开发布；从 0.4.8 更新会同时获得算式纸、Control 双击直达与导入键位先审后用

修复

  • 修复停止录制时的竞态崩溃、菜单栏状态贴片可能令 CPU 满载、录制确认条遮住系统权限弹窗
  • 轻点 fn 键不再误唤出面板
