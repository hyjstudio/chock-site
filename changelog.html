0.4.7 · 2026-07-15

  • 面板即计算器：搜索框直接输入算式（23*4.5+8、200+10%、200的10%），结果内联出现，回车即复制；纯数字和应用名不受影响，搜索照旧
  • 计算器与剪贴板联动：刚复制过数字，以运算符开头（如 *1.13）自动以它为首操作数；复制的结果自动进剪贴板历史；ans 连续计算
  • 单位换算：10ft in m、5斤=克、100°F转°C、128GB=MB，支持斤、两、亩、里等本土单位
  • 人民币大写金额：「12345.67 大写」→ 壹万贰仟叁佰肆拾伍元陆角柒分
  • 键位网格直接起算：按未绑定的数字或运算符键，自动进搜索框接着写算式
  • 行为变更：搜索框为空时按数字键现在开始输入，不再直接运行第 N 项；方向键选中某行后回车永远执行该行
  • 到期表单搬进真正的表单窗：回车保存、Esc 取消；日期只保留一组大号上下箭头
  • 空状态会教你了：粘贴面板、剪贴板历史、到期追踪的空白页升级为图标+说明+下一步提示

0.4.6 · 2026-07-15

  • 新装机全局快捷键开箱即用：Ctrl+A 截图、Ctrl+X 剪贴板历史、Ctrl+B 粘贴面板、Ctrl+I 截图取字、Ctrl+J 钉图显隐、Ctrl+M 标注默认全局生效；已有配置不受更新影响，「恢复默认键位」可取回全新状态
  • 到期提醒点了有去处：点击到期通知直达到期追踪并定位对应条目；提醒提前量可按条目自定义（当天/1天/3天/1周/1个月）；单条订阅可静音；新增「半年」周期
  • 到期追踪添加体验重做：先选周期和开始日，到期日自动算出；日期两侧新增大号「±1 天」按钮
  • 资源管理：内存管理升级为内存 | CPU 双页签，按应用 CPU 占用一目了然（口径同活动监视器）
  • Markdown 编辑器补全文件闭环：动作直接打开编辑器；新增「打开…」「另存为…」「最近」；标题栏挂原生文件图标；Finder「打开方式」不再覆盖未保存内容
  • 粘贴面板首开引导：首次打开说明「点击条目即粘贴」，未授权辅助功能时「去开启」直达系统设置
  • 截图提示音开关搬到手边：状态栏菜单新增开关，与防休眠、隐藏桌面并排
  • 剪贴板历史容量 50 → 200：搜索长尾不再被截断；慢按 Ctrl+X 不再向搜索框漏字母
  • 界面细节：设置五页统一居中内容列；三个主窗口大标题统一字级；混合组补强（单选链接自动存为链接绑定）

0.4.4 · 2026-07-13

0.4.3 · 2026-07-13

  • 唤出面板不再挡住正式窗口：从面板打开设置、剪贴板历史等界面时，会先同步收起面板，再呈现目标窗口

  • 内存管理可以安全退出应用：内存面板只结束当前看到的具体应用实例；应用重启后不会继承旧状态，并继续保护 Finder、Dock 等系统进程
  • 截图细节更可靠：修复极小窗口和奇偶尺寸窗口的边角采样；全局选区、录屏与翻译覆盖层统一使用同一套显示器解析规则
  • 交互路径更轻：取色改为单像素采样，内存窗口不再连带采集无关的 CPU、GPU、磁盘和网络指标；同时修复系统状态采样的 Mach 端口权泄漏
  • 发行身份闭环：App、DMG、ZIP、Appcast、Sparkle 签名密钥和源码提交必须一致，任一层不匹配都会阻止发布
