0.4.9 · 2026-07-16

计算器变成一张真正好用的算式纸。计算过程可以直接用鼠标拖选复制，也可以一键复制全部；常用的 ＋、−、×、÷、＝ 放在手边，不熟键位也能直接开始。

新功能

  • Control 双击直达：左、右 Control 可以分别配置，和 Command、Option、Shift 一样双击唤起应用或动作

改进

  • 算式纸的长结果自动换行，人民币大写金额等内容不再被截断；历史支持复制全部与清空
  • 导入键位配置会统一检查动作、触发键与风险项，Shell 脚本等能力必须确认后才会注册

0.4.8 · 2026-07-15

截图,现在选完区就能直接标注。框选之后,工具条就在选区上——矩形、椭圆、箭头、直线、画笔、文字、序号、高亮、马赛克随手用,画完按确认,拿到的就是成品,不再需要打开第二个窗口。马赛克遮住的内容,取字和翻译也同样看不到。

新功能

  • 计算器窗口:可绑键位,状态栏和面板底部都有入口;大字结果、算式回显、可点击复制的历史,ans 连算、单位换算、人民币大写金额都在
  • 剪贴板钉到屏幕:历史里右键任意条目——文字变成置顶浮动卡片,图片变成钉图;「显隐钉图」一键清场或召回全部
  • 粘贴面板可以钉住:点亮右上角图钉后,去目标应用点击定位光标面板也不收起,粘贴、点位、再粘贴一气呵成;Esc 收起
  • 标注新工具:椭圆、直线上架;矩形、椭圆、直线、箭头都可切换虚线描边
  • Markdown 公式:行内 $E=mc^2$ 与块级 $$…$$ 都能写,预览和 HTML/PDF 导出全保留;在本机渲染,不联网
  • Shell 脚本键位:右键键位可绑定你自己的 .sh/.command,静默或在终端执行;失败会弹出提示,绝不静默
  • 最近选区召回:截图时按 ⌘←/⌘→ 在最近 5 个选区之间切换,跨重启保留

改进

  • 到期卡片按分类显示图标(服务器、域名、流媒体、证件…),一眼可扫
  • 马赛克落笔即显示真实像素化效果,所见即所得
  • 键位格子可选显示动作名称(设置 → 唤出),默认保持安静网格
  • 刘海机型上,截图和录屏的顶部提示条自动避开刘海
  • 长时间运行更省内存:菜单栏刷新、剪贴板与截图管线均已瘦身

修复

  • 马赛克打码位置上下颠倒:此前打码会落在所画区域的垂直镜像处,导出同样错位——用马赛克遮过敏感信息的旧图,建议复查一遍

0.4.7 · 2026-07-15

在搜索框里直接算数。输入 23*4.5+8、200+10%、200的10%,结果立刻出现,回车即复制;单位换算(10ft in m、5斤=克、128GB=MB)和人民币大写金额(12345.67 大写 → 壹万贰仟…)同样一句直达。

新功能

  • 计算与剪贴板联动:刚复制过数字,以运算符开头(如 *1.13)自动接着它算;ans 引用上一次结果连续计算
  • 键位网格直接起算:按未绑定的数字或运算符键,自动进入搜索框接着写算式

改进

  • 到期项目的添加和编辑搬进真正的表单窗口:回车保存、Esc 取消,日期用一组大号上下箭头调整
  • 粘贴面板、剪贴板历史、到期追踪的空白页会告诉你下一步怎么做(如「⌘1–9 秒贴前九条」)

修复

  • 搜索框为空时按数字键改为开始输入,不再直接运行第 N 项;有文字后按数字仍是快捷运行
  • 方向键选中某一行后,回车永远执行该行,不会被计算结果抢走

  • 剪贴板历史容量 50 → 200,搜索长尾不再被截断
  • 粘贴面板第一次打开会说明「点击条目即粘贴」,未授权时提供「去开启」直达系统设置
